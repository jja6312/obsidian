#### 0. 베어메탈

#### 1. 📌한줄 정의

- 가상화 계층 없이 물리 하드웨어에 직접 액세스할 수 있는 단일 테넌트 전용 서버

#### 2. 쉽게 설명

- 하이퍼바이저나 가상화 소프트웨어 없이 직접 하드웨어를 사용하는 서버
- 마치 자신의 데이터센터에 있는 물리 서버처럼 완전한 제어와 성능을 제공
- 높은 성능이 필요한 워크로드나 라이센스 요구사항이 있는 애플리케이션에 적합

#### 3. 🔍 특징

- 최고 수준의 성능과 격리성
- 하드웨어 수준의 완전한 제어 가능
- 단일 테넌트 전용으로 보안성 우수
- 예측 가능한 일관된 성능
- 특정 워크로드에 최적화 가능

#### 4. tree 구조(부모/형제/자식)

- 부모 : OCI Compute
- 형제
  - VM
  - DVM
- 자식
  - Standard Bare Metal
  - DenselO Bare Metal
  - GPU Bare Metal
  - HPC Bare Metal

#### 5. 예시

- 사용 사례
  - 대규모 DB 워크로드
  - HPC 애플리케이션
  - 빅데이터 분석
  - AI/ML 워크로드
  - Oracle RAC 구성
  - 레거시 애플리케이션 마이그레이션
